## The Future of Employment
### 前言
- 隨著時代變遷，科技網路快速進步使得工作競爭更加激烈，無論是人與人之間的相互取代，亦或是因為AI科技進展而產生的科技取代，都慢慢成為社會失業率上升的影響因素。

- 想知道自己的工作在未來是不是很有可能因為科技的進步而被機器替代，藉由這次研究主要探討現有的就業結構狀況，以及哪些工作在近來有被取代的風險及可能
